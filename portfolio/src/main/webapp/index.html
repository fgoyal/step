MechMania 2020

MechMania is a 24-hour Game AI hackathon and a part of the Reflections Projections Conference hosted by ACM @ UIUC. Each year, the MechMania staff build a game in relative secrecy. Competitors have 24 hours in which to implement a bot to play the game. These bots face off bots built by other teams and the team that builds the best bots win.
